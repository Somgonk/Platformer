# Platformer Game

---

## Clone project
```
git clone https://github.com/Somgonk/Platformer.git --recurse-submodules
```

## Compile project

Step 1: Make build directory
```
mkdir build
cd build
```

Step 2: Compile makefile from cmake
```
cmake ..
```

Step 3: Compile project
```
make
```

Step 4: Run program
```
./platformer
```
